## Quick Start

```bash
# Add dependency
dart pub add design_builder --dev

# Create your token files
echo '{"light": {"color": {"primary": {"$type": "color", "$value": "#1A1A2E"}}}}' > lib/tokens.json

# Generate code
dart run build_runner build
```
